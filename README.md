## Условие задачи

Определить структурный тип, описывающий расписание маршрутов электропоездов (название маршрута, пункт отправления, пункт прибытия, время отправления, время прибытия, количество вагонов, тип электропоезда: товарный, пассажирский, ремонтный и др.).

Заполнить структурный массив 20-ю записями.

- Переписать из исходного массива в другой массив информацию только о тех грузовых маршрутах, что идут в приморские города. Затем новый массив отсортировать по количеству вагонов.
- Вывести все данные по конкретному электропоезду.
- Вывести 3 самых длинных пассажирских поезда.
- Реализовать функцию изменения данных о маршруте.
- Реализовать вывод отфильтрованных данных в виде оберточной функции.
- Считать из текстового файла данные числового поля (количество вагонов) и обновить массив структур.
- Сделать чтение/запись структуры в бинарный файл, оформить в виде подпрограмм.

---


## Формат файла train_numcars.txt

Каждая строка — название маршрута и новое количество вагонов через пробел:

```
Грузовой1 30
Грузовой2 25
Грузовой3 17
Пасс1 12
Пасс4 20
```

Файл должен лежать в той же папке, что и скомпилированный исполняемый файл.

---

## Примечания

- Бинарный файл `routes.bin` создаётся автоматически при первом запуске программы.
- При повторном запуске программа обнаружит `routes.bin` и прочитает данные из него.
- Сортировка реализована алгоритмом пузырька.
- Ввод данных осуществлён инициализацией массива структур внутри программы.
